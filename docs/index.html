Curso de Análisis Estadístico en R Nivel Intermedio



¡Bienvenidos!

Este es un curso de análisis estadístico en R utilizando R Studio. Utilizaremos el formato R Markdown para crear nuestros códigos y presentar nuestros resultados de una forma ordenada. Analizaremos nuestros datos mediante modelos lineales generales, modelos lineales generalizados (glm), y análisis multivariado.

¡Comencemos!
